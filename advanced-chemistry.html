化学1

方程式计算

一定质量的碳酸钙与足量稀盐酸反应，生成二氧化碳4.4g。求参加反应的碳酸钙质量。

  1. 写方程式：CaCO3+2HCl=CaCl2+H2O+CO2↑。
  2. 相对质量关系：CaCO3为100，CO2为44。
  3. 设碳酸钙质量为x，则100/44=x/4.4。
  4. 解得x=10g。

答案：参加反应的碳酸钙质量为10g。

溶液稀释

把80g质量分数为15%的氯化钠溶液稀释成6%，需要加水多少克？

  2. 原溶质质量=80×15%=12g。
  3. 稀释后溶液质量=12/6%=200g。
  4. 需要加水=200-80=120g。

答案：需要加水120g。

化学3

金属活动性推理

将铁片放入硫酸铜溶液中，溶液蓝色变浅并有红色固体析出；将铜片放入硝酸银溶液中，有银白色固体析出。判断Fe、Cu、Ag活动性顺序。

  1. 铁能把铜从硫酸铜中置换出来，说明Fe比Cu活泼。
  2. 铜能把银从硝酸银中置换出来，说明Cu比Ag活泼。
  3. 把两个结论连接起来，得Fe>Cu>Ag。
  4. 写方程式可辅助说明：Fe+CuSO4=FeSO4+Cu，Cu+2AgNO3=Cu(NO3)2+2Ag。

答案：金属活动性顺序为Fe>Cu>Ag。

化学4

酸碱中和图象

向氢氧化钠溶液中逐滴加入稀盐酸，溶液pH由13逐渐降到7以下。判断恰好完全反应的位置，并说明溶液中主要溶质变化。

  1. 氢氧化钠溶液呈碱性，初始pH大于7。
  2. 加入盐酸发生中和反应：NaOH+HCl=NaCl+H2O。
  3. 恰好完全反应时溶液近似中性，pH=7。
  4. pH小于7后，盐酸过量，溶质为NaCl和HCl；pH=7时主要溶质为NaCl。

答案：pH=7处恰好完全反应；此时主要溶质为NaCl。

化学5

物质推断

A、B、C均为初中常见物质。A是黑色固体，通入CO加热可得到红色金属B，同时生成能使澄清石灰水变浑浊的气体C。推断A、B、C。

  1. 能使澄清石灰水变浑浊的气体是CO2，所以C为二氧化碳。
  2. CO具有还原性，能把金属氧化物还原成金属。
  3. 生成红色金属，初中常见为铜。
  4. 黑色固体A应为氧化铜CuO，反应为CO+CuO=Cu+CO2。

答案：A为CuO，B为Cu，C为CO2。

化学6

除杂推理

NaCl固体中混有少量Na2CO3。若要得到较纯NaCl，可加入哪种试剂？写出原理并说明为什么不能过量。

  1. 杂质Na2CO3能与稀盐酸反应生成NaCl、水和CO2。
  2. 加入适量稀盐酸可除去碳酸钠，同时生成目标物NaCl。
  3. 方程式：Na2CO3+2HCl=2NaCl+H2O+CO2↑。
  4. 若盐酸过量，会引入新杂质HCl，所以必须适量。

答案：加入适量稀盐酸；不能过量，避免引入HCl。

化学7

流程转化

石灰石经高温煅烧得到氧化钙，氧化钙与水反应得到氢氧化钙，氢氧化钙吸收二氧化碳又生成碳酸钙。写出三个方程式。

  1. 石灰石主要成分为CaCO3，高温分解：CaCO3=CaO+CO2↑。
  2. 氧化钙与水反应：CaO+H2O=Ca(OH)2。
  3. 氢氧化钙与二氧化碳反应：Ca(OH)2+CO2=CaCO3↓+H2O。
  4. 检查每个方程式是否配平，沉淀和气体符号是否合理。

答案：三个方程式依次为CaCO3分解、CaO化合、Ca(OH)2与CO2反应。

化学8

质量守恒

某反应A+2B=3C。已知8g A与足量B完全反应生成24g C，求参加反应的B质量。

  1. 根据质量守恒定律，反应物总质量等于生成物总质量。
  2. 生成C质量为24g，A质量为8g。
  3. 参加反应的B质量=24-8=16g。
  4. 题中“足量B”表示B有剩余，但参加反应的只计算16g。

答案：参加反应的B质量为16g。

溶解度判断

20℃时某物质溶解度为30g。将40g该物质加入100g水中充分搅拌，求所得溶液质量和溶质质量分数。

  1. 溶解度30g表示20℃时100g水最多溶解30g该物质。
  2. 加入40g时只有30g溶解，剩余10g不溶。
  3. 所得溶液质量=100+30=130g。
  4. 溶质质量分数=30/130×100%≈23.1%。

答案：溶液质量130g，溶质质量分数约23.1%。

化学10

离子共存

某无色溶液中可能含有H+、Cu2+、Cl-、CO3^2-、Na+。根据“无色”和“加入稀盐酸有气泡”判断一定不存在和可能存在的离子。

  1. 无色溶液中一般不含Cu2+，因为Cu2+溶液常呈蓝色。
  2. 加入稀盐酸有气泡，说明原溶液可能含CO3^2-，生成CO2。
  3. 若溶液中大量H+与CO3^2-会反应，通常不能大量共存。
  4. Na+、Cl-一般可共存。

答案：一定不含Cu2+；有CO3^2-证据；H+与CO3^2-不能大量共存，Na+、Cl-可能存在。
