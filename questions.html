数学大题10道

数学 1 / 方程应用

打折销售与一元二次方程

某文具店购进一批笔记本，每本进价8元。若每本售价12元，每天可售出60本；经调查，售价每降低0.5元，每天可多售出10本。若要求每天利润为240元，且尽量让学生少花钱，售价应定为多少元？

  1. 设每本降价 x 元，则售价为 12-x 元，每本利润为 (12-x)-8=4-x 元。
  2. 售价每降低0.5元，多售10本，所以每降低1元，多售20本；每天销量为 60+20x 本。
  3. 根据“总利润=单件利润×销量”列方程：(4-x)(60+20x)=240。
  4. 展开：240+80x-60x-20x^2=240，整理得 -20x^2+20x=0。
  5. 提取公因式：-20x(x-1)=0，所以 x=0 或 x=1。
  6. 题目要求“尽量让学生少花钱”，应选择降价更多的 x=1。
  7. 售价为 12-1=11 元。

答案：售价应定为11元。

拿分提醒：应用题最后要回到问题本身。两个解都能满足利润时，看题目有没有“尽量便宜、尽量多卖”等条件。

数学 2 / 一次函数

两种收费方案比较

某体育馆有两种购票方案。方案A：每次20元；方案B：先交会员费60元，以后每次12元。设去体育馆 x 次，总费用分别为 yA、yB。1. 写出两个函数关系式；2. 去多少次时两方案费用相同；3. 若计划去12次，选哪种更省钱？

  1. 方案A没有固定费用，每次20元，所以 yA=20x。
  2. 方案B先交60元，再按每次12元收费，所以 yB=60+12x。
  3. 费用相同时，20x=60+12x，得 8x=60，x=7.5。
  4. 次数必须是整数。若 x≤7，方案A费用低；若 x≥8，方案B费用低。
  5. 计划去12次：yA=20×12=240元；yB=60+12×12=204元。

答案：yA=20x，yB=60+12x；7.5次时相同；12次选方案B，省36元。

拿分提醒：函数应用题要把“固定费用”和“每次费用”分开，交点表示两种方案一样贵。

数学 3 / 几何证明

平行线与相似三角形

如图，在三角形ABC中，DE∥BC，D在AB上，E在AC上。已知 AD=4，DB=6，AC=15。1. 证明三角形ADE相似三角形ABC；2. 求AE；3. 若DE=6，求BC。

  1. 因为 DE∥BC，所以 ∠ADE=∠ABC，∠AED=∠ACB。
  2. 又因为 ∠A 是公共角，所以三角形ADE与三角形ABC满足两角分别相等。
  3. 因此 △ADE∽△ABC。
  4. AB=AD+DB=4+6=10。由相似可得 AD/AB=AE/AC=DE/BC。
  5. 代入 AD/AB=4/10=2/5，所以 AE/15=2/5，得 AE=6。
  6. 又 DE/BC=2/5，6/BC=2/5，得 BC=15。

答案：△ADE∽△ABC；AE=6；BC=15。

拿分提醒：证明相似时不要只写“因为平行所以相似”，要写出两组对应角相等。

数学 4 / 圆综合

切线、勾股定理与面积

圆O的半径为5，PA、PB分别切圆O于A、B两点，OP=13。1. 求PA；2. 判断四边形OAPB中∠OAP的大小；3. 求三角形OAP的面积。

  1. 连接OA。半径垂直于切线，所以 OA⊥PA，∠OAP=90°。
  2. 在直角三角形OAP中，OP是斜边，OA=5，OP=13。
  3. 由勾股定理：PA^2=OP^2-OA^2=13^2-5^2=169-25=144。
  4. 所以 PA=12。
  5. 三角形OAP面积 S=1/2×OA×PA=1/2×5×12=30。

答案：PA=12；∠OAP=90°；面积为30。

拿分提醒：圆切线大题的第一步通常是“连接半径”，这一步经常值1分。

数学 5 / 二次函数

抛物线与坐标轴交点

已知抛物线 y=-x^2+4x+5。1. 求顶点坐标；2. 求它与x轴的交点坐标；3. 当 y≥0 时，求x的取值范围。

  1. 配方：y=-(x^2-4x)+5=-[(x-2)^2-4]+5=-(x-2)^2+9。
  2. 所以顶点坐标为 (2,9)，对称轴为 x=2。
  3. 与x轴交点令 y=0：-x^2+4x+5=0，等价于 x^2-4x-5=0。
  4. 因式分解：(x-5)(x+1)=0，得 x=5 或 x=-1。
  5. 交点为 (-1,0)、(5,0)。抛物线开口向下，图象在x轴上方或x轴上时，x在两个根之间。

答案：顶点(2,9)；交点(-1,0)、(5,0)；y≥0时 -1≤x≤5。

拿分提醒：二次函数大题先配方，顶点、对称轴、最大值一次解决。

数学 6 / 统计综合

样本数据与决策

某班抽取10名学生一分钟跳绳成绩：128，132，135，140，140，145，148，150，150，152。1. 求平均数和中位数；2. 若学校规定150个及以上为优秀，估计全班50人中优秀人数；3. 说明用样本估计总体时要注意什么。

  1. 总数为 1420，所以平均数=1420÷10=142。
  2. 10个数据已经按从小到大排列，中位数为第5个和第6个的平均数：(140+145)/2=142.5。
  3. 优秀的有150、150、152，共3人，样本优秀率为 3/10=30%。
  4. 估计全班优秀人数：50×30%=15人。
  5. 样本估计总体时，样本应具有代表性，人数不能太少，抽样要尽量随机。

答案：平均数142，中位数142.5；估计优秀约15人；注意样本代表性和随机性。

拿分提醒：统计大题最后一问常考“样本代表性”，这句话要会写。

数学 7 / 解直角三角形

测高测距应用

小明在距教学楼底部30 m处测得楼顶仰角为37°，小明眼睛离地1.5 m。已知 tan37°≈0.75，求教学楼高度。

  1. 先画直角三角形。水平距离30 m是邻边，楼顶到眼睛水平线的高度差设为 h。
  2. 由正切定义，tan37°=h/30。
  3. h=30×0.75=22.5 m。
  4. 教学楼总高度=高度差+眼睛高度=22.5+1.5=24 m。
  5. 答题时要写“约为”，因为 tan37° 是近似值。

答案：教学楼高度约为24 m。

拿分提醒：测高题别漏“眼睛高度”，这是河北中考应用题常见扣分点。

数学 8 / 概率综合

列表法求两次随机事件概率

袋中有编号1、2、3的三个小球，除编号外完全相同。先摸出一个球，记录编号后放回，再摸出一个球。1. 用列表法列出所有结果；2. 求两次编号之和为偶数的概率。

  1. 因为摸后放回，所以第二次仍有1、2、3三种可能。
  2. 所有等可能结果共有 3×3=9 种：(1,1)、(1,2)、(1,3)、(2,1)、(2,2)、(2,3)、(3,1)、(3,2)、(3,3)。
  3. 和为偶数的条件是“两数同奇偶”：奇+奇或偶+偶。
  4. 符合的有 (1,1)、(1,3)、(2,2)、(3,1)、(3,3)，共5种。
  5. 概率 P=符合结果数/全部结果数=5/9。

答案：共有9种等可能结果；编号和为偶数的概率为5/9。

拿分提醒：放回是9种，不放回是6种；审题的“放回”两个字很关键。

数学 9 / 几何综合

矩形中的折叠问题

矩形ABCD中，AB=6，AD=8。将点D沿折痕AE折到点F，F落在BC上，E在CD上。求BF的长。

  1. 折叠问题的核心是对应线段相等：AD=AF，所以 AF=8。
  2. 因为F在BC上，AB⊥BC，所以三角形ABF是直角三角形。
  3. 在直角三角形ABF中，AF是斜边，AB=6，AF=8。
  4. 由勾股定理：BF^2=AF^2-AB^2=8^2-6^2=64-36=28。
  5. 所以 BF=sqrt(28)=2sqrt(7)。

答案：BF=2sqrt(7)。

拿分提醒：折叠题先写对应边相等，再找直角三角形用勾股定理。

数学 10 / 综合压轴入门

一次函数与面积最值

直线 y=-x+6 与坐标轴交于A、B两点，点P在第一象限的线段AB上，过P作PC⊥x轴，垂足为C。设P点横坐标为x。1. 写出P点坐标；2. 用x表示三角形POC面积；3. 求面积最大值。

  1. 点P在线段AB上，且横坐标为x，所以纵坐标 y=-x+6，P(x, -x+6)。
  2. 第一象限内，0<x<6。
  3. 三角形POC的底 OC=x，高 PC=-x+6。
  4. 面积 S=1/2×x×(-x+6)= -1/2x^2+3x。
  5. 配方：S= -1/2(x^2-6x)= -1/2[(x-3)^2-9]= -1/2(x-3)^2+4.5。
  6. 因为 -1/2(x-3)^2≤0，所以当 x=3 时，S最大，最大值为4.5。

答案：P(x, -x+6)；S=-1/2x^2+3x；最大面积4.5。

拿分提醒：压轴题先拿前两问。把点坐标和面积表达式写出来，已经有不少过程分。

物理 1 / 力学计算

运动、重力与功的综合

小车质量为20 kg，在水平地面上匀速前进40 m，用时10 s。若小车受到的阻力为重力的0.05倍，g取10 N/kg。求：1. 小车速度；2. 小车重力；3. 推力做的功。

  1. 速度公式 v=s/t，代入 s=40 m，t=10 s，得 v=4 m/s。
  2. 重力 G=mg=20×10=200 N。
  3. 小车匀速直线运动，水平方向受力平衡，所以推力 F 等于阻力 f。
  4. 阻力 f=0.05G=0.05×200=10 N，因此推力 F=10 N。
  5. 推力做功 W=Fs=10×40=400 J。

答案：速度4 m/s；重力200 N；推力做功400 J。

拿分提醒：看到“匀速”，马上想到“受力平衡”，推力等于阻力。

物理 2 / 压强浮力

液体压强与浮力

一物块体积为3×10^-4 m^3，完全浸没在水中，物块上表面距水面0.2 m，下表面距水面0.5 m。水的密度1.0×10^3 kg/m^3，g取10 N/kg。求：1. 下表面受到的液体压强；2. 物块所受浮力。

  1. 液体压强公式 p=rho gh。下表面深度 h=0.5 m。
  2. p=1.0×10^3×10×0.5=5.0×10^3 Pa。
  3. 浮力公式 F浮=rho液gV排。
  4. 物块完全浸没，所以 V排=V物=3×10^-4 m^3。
  5. F浮=1.0×10^3×10×3×10^-4=3 N。

答案：5.0×10^3 Pa；3 N。

拿分提醒：液体压强看深度，不看容器形状；浮力看排开液体体积。

物理 3 / 杠杆机械

杠杆平衡与机械效率

用一根轻质杠杆提升重物。重物重120 N，重物到支点距离0.2 m，人的拉力作用点到支点距离0.6 m。1. 杠杆平衡时拉力多大；2. 若把重物提高0.1 m，克服重力做多少有用功？

  1. 杠杆平衡条件：F1L1=F2L2。
  2. 重物一侧力矩为 120×0.2=24 N·m。
  3. 设人的拉力为F，则 F×0.6=24，F=40 N。
  4. 有用功是克服重力做功，W有=Gh。
  5. W有=120×0.1=12 J。

答案：拉力40 N；有用功12 J。

拿分提醒：杠杆题先找支点，力臂是支点到力的作用线的距离。

物理 4 / 电学计算

串联电路与电功率

如图，R1=10 ohm，R2=20 ohm 串联接在电压为6 V的电源两端。求：1. 电路总电阻；2. 电流；3. R2消耗的电功率。

  1. 串联电路总电阻 R=R1+R2=10+20=30 ohm。
  2. 由欧姆定律 I=U/R，得 I=6/30=0.2 A。
  3. 串联电路电流处处相等，所以通过R2的电流也是0.2 A。
  4. R2的电功率 P2=I^2R2=(0.2)^2×20=0.04×20=0.8 W。
  5. 也可先算 U2=IR2=4 V，再用 P=UI=4×0.2=0.8 W。

答案：总电阻30 ohm；电流0.2 A；R2功率0.8 W。

拿分提醒：电功率公式很多，基础薄弱时优先用 P=UI；知道电阻和电流时用 P=I^2R。

物理 5 / 滑变电路

滑动变阻器变化分析

定值电阻R与滑动变阻器串联，电压表测R两端电压，电流表测电路电流。电源电压不变。当滑片向右移动时，接入电路的滑动变阻器电阻变大。判断电流表示数、电压表示数如何变化，并说明理由。

  1. 先判断电路：定值电阻R与滑动变阻器串联。
  2. 滑片向右，滑动变阻器接入电阻变大，所以电路总电阻 R总 变大。
  3. 电源电压U不变，由 I=U/R总 可知，电路电流变小，所以电流表示数变小。
  4. 电压表测定值电阻R两端电压，R不变，电流I变小。
  5. 由 UR=IR 可知，定值电阻两端电压变小，所以电压表示数变小。

答案：电流表示数变小，电压表示数变小。

拿分提醒：滑变题固定三步：总电阻怎么变、电流怎么变、被测电压怎么变。

物理 6 / 电热安全

家庭电路与电能计算

某电热水壶标有“220 V 1100 W”。1. 正常工作时电流约多大；2. 正常工作6 min消耗多少电能；3. 为什么不能用湿手触摸开关？

  1. 正常工作时，P=UI，所以 I=P/U=1100/220=5 A。
  2. 6 min=0.1 h。用电能单位 kW·h 计算更方便，1100 W=1.1 kW。
  3. W=Pt=1.1×0.1=0.11 kW·h。
  4. 若用焦耳：6 min=360 s，W=Pt=1100×360=3.96×10^5 J。
  5. 湿手电阻较小，人体更容易有较大电流通过，可能触电，所以不能湿手触摸开关。

答案：电流5 A；电能0.11 kW·h，即3.96×10^5 J；湿手易触电。

拿分提醒：家庭电路题除了计算，还常考安全用电原因，要写“人体电阻小、电流大”。

物理 7 / 热学计算

比热容与热效率

用燃气灶把2 kg水从20℃加热到70℃，水的比热容为4.2×10^3 J/(kg·℃)。若燃气完全燃烧放出5.0×10^5 J热量，求：1. 水吸收热量；2. 加热效率。

  1. 温度变化 Δt=70-20=50℃。
  2. 水吸收热量 Q吸=cmΔt=4.2×10^3×2×50。
  3. Q吸=4.2×10^5 J。
  4. 热效率 η=Q吸/Q放×100%。
  5. η=4.2×10^5 / 5.0×10^5 ×100%=84%。

答案：水吸收4.2×10^5 J；加热效率84%。

拿分提醒：比热容题先算温度变化，不要把末温70℃直接代入。

物理 8 / 光学实验

凸透镜成像规律

在探究凸透镜成像规律实验中，凸透镜焦距为10 cm。蜡烛到透镜距离为25 cm。1. 像的性质是什么；2. 光屏应放在哪个范围；3. 若把蜡烛向透镜靠近到15 cm，像如何变化？

  1. 焦距 f=10 cm，2f=20 cm。初始物距 u=25 cm，大于2f。
  2. 当 u>2f 时，成倒立、缩小的实像。
  3. 此时像距 v 在 f 和 2f 之间，即10 cm到20 cm之间，光屏应放在透镜另一侧这个范围内调节。
  4. 当蜡烛靠近到15 cm时，物距在 f 和 2f 之间。
  5. 此时成倒立、放大的实像，像距大于2f，即大于20 cm；像变大，光屏要远离透镜。

答案：先成倒立缩小实像，光屏在10到20 cm间；物距15 cm时成倒立放大实像，光屏远离透镜。

拿分提醒：凸透镜只要背住三个区间：u>2f、u=2f、f<u<2f。

物理 9 / 实验探究

伏安法测电阻

实验小组用伏安法测未知电阻Rx。某次电压表示数2.4 V，电流表示数0.30 A。1. 求Rx；2. 滑动变阻器有什么作用；3. 为什么要多次测量求平均值？

  1. 根据欧姆定律变形，R=U/I。
  2. 代入 U=2.4 V，I=0.30 A，得 R=2.4/0.30=8 ohm。
  3. 滑动变阻器的作用有两个：闭合开关前接入最大阻值以保护电路；实验中改变电路电流和电阻两端电压。
  4. 多次测量可得到多组U、I，分别算出多个R值。
  5. 对多个R值取平均，可以减小偶然误差，使结果更接近真实值。

答案：Rx=8 ohm；滑动变阻器保护电路并改变电压电流；多次测量求平均可减小误差。

拿分提醒：实验题问“为什么多次测量”，测定值通常答“减小误差”，找规律通常答“避免偶然性”。

物理 10 / 综合实验

探究影响滑动摩擦力大小的因素

用弹簧测力计水平拉木块，使木块在水平桌面上做匀速直线运动。第一次木块单独运动，测力计示数1.2 N；第二次在木块上加砝码，示数1.8 N；第三次把木块放到更粗糙的毛巾面上，示数2.4 N。回答：1. 为什么要匀速拉动；2. 比较前两次说明什么；3. 比较第一次和第三次说明什么。

  1. 木块匀速直线运动时，水平方向受力平衡，拉力大小等于滑动摩擦力大小。
  2. 第一次和第二次接触面粗糙程度相同，第二次压力更大，摩擦力由1.2 N变为1.8 N。
  3. 因此可得：接触面粗糙程度相同时，压力越大，滑动摩擦力越大。
  4. 第一次和第三次压力相同，但第三次接触面更粗糙，摩擦力由1.2 N变为2.4 N。
  5. 因此可得：压力相同时，接触面越粗糙，滑动摩擦力越大。
  6. 本实验用到控制变量法：研究一个因素时，其余因素保持不变。

答案：匀速时拉力等于摩擦力；压力越大摩擦力越大；接触面越粗糙摩擦力越大。

拿分提醒：物理实验大题常考“控制变量法”，回答时要点名“只改变一个因素”。
