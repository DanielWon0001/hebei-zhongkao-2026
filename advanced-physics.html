物理1

串联电路动态分析

如图，电源电压恒为6V，定值电阻R=10Ω与滑动变阻器串联，电压表测R两端电压。滑片向右移动时，变阻器接入阻值由5Ω变为20Ω。求移动前后电流、电压表示数和R的电功率变化。

  1. 移动前总电阻R总=10+5=15Ω，电流I1=6/15=0.4A。
  2. R两端电压U1=I1R=0.4×10=4V，功率P1=I1^2R=1.6W。
  3. 移动后总电阻R总=10+20=30Ω，电流I2=6/30=0.2A。
  4. R两端电压U2=0.2×10=2V，功率P2=0.2^2×10=0.4W。

答案：电流0.4A变0.2A；电压4V变2V；功率1.6W变0.4W，均减小。

物理2

浮力与沉浮

体积为4×10^-4m³的物块重5N，完全浸没在水中。求浮力，并判断松手后物块上浮、悬浮还是下沉。g取10N/kg，ρ水=1.0×10³kg/m³。

  1. 完全浸没时V排=V物=4×10^-4m³。
  2. F浮=ρ水gV排=1.0×10³×10×4×10^-4=4N。
  3. 比较浮力和重力：F浮=4N
  4. 合力方向向下，所以松手后物块下沉。

答案：浮力4N，物块下沉。

物理3

速度图象

某车0-4s速度从0均匀增至8m/s，4-10s保持8m/s。求10s内路程和平均速度。

  1. v-t图象面积表示路程。
  2. 0-4s为三角形面积：1/2×4×8=16m。
  3. 4-10s为矩形面积：6×8=48m。
  4. 总路程64m，平均速度v=s/t=64/10=6.4m/s。

答案：路程64m，平均速度6.4m/s。

物理4

压强与压力

水深0.5m，容器底面积0.03m²。求水对底部压强和压力。ρ水=1.0×10³kg/m³，g取10N/kg。

  1. 液体压强只与液体密度、g和深度有关。
  2. p=ρgh=1.0×10³×10×0.5=5.0×10³Pa。
  3. 压力F=pS=5.0×10³×0.03=150N。

答案：压强5.0×10³Pa，压力150N。

物理5

机械功率

重为300N的物体在拉力作用下沿水平面匀速移动20m，摩擦力为60N，用时10s。求拉力做功和功率。

  1. 匀速直线运动，水平方向受力平衡，拉力F=f=60N。
  2. 拉力做功W=Fs=60×20=1200J。
  3. 功率P=W/t=1200/10=120W。

答案：做功1200J，功率120W。

电热综合

标有220V 1100W的电热器正常工作12min。求正常电流、消耗电能，并说明为什么同一插座不能同时接太多大功率用电器。

  1. 正常电流I=P/U=1100/220=5A。
  2. 12min=0.2h，1100W=1.1kW，电能W=Pt=1.1×0.2=0.22kW·h。

物理7

热效率

2kg水从20℃升至70℃，燃料完全燃烧放热5.0×10^5J。求水吸收热量和加热效率。c水=4.2×10³J/(kg·℃)。

  1. 温度变化Δt=70-20=50℃。
  2. 水吸热Q=cmΔt=4.2×10³×2×50=4.2×10^5J。
  3. 效率η=Q吸/Q放×100%=4.2×10^5/5.0×10^5×100%=84%。

答案：吸热4.2×10^5J，效率84%。

物理8

凸透镜成像

凸透镜焦距10cm，物体距透镜15cm。判断像的性质、像距范围；若物体逐渐远离到25cm，像如何变化。

  1. f=10cm，2f=20cm。u=15cm，在f和2f之间。
  2. 此时成倒立、放大的实像，像距v>2f，即大于20cm。
  3. 当物体远离到25cm时，u>2f，成倒立、缩小的实像，像距在f和2f之间。
  4. 因此像由放大变为缩小，光屏应向靠近透镜方向移动。

答案：15cm时倒立放大实像，像距大于20cm；25cm时倒立缩小实像，像距10-20cm。

物理9

并联电路

R1=12Ω，R2=6Ω并联接在6V电源上。求两支路电流、总电流和总功率。

  1. 并联电路各支路电压相等，均为6V。
  2. I1=U/R1=6/12=0.5A，I2=6/6=1A。
  3. 总电流I=I1+I2=1.5A。
  4. 总功率P=UI=6×1.5=9W。

答案：I1=0.5A，I2=1A，总电流1.5A，总功率9W。

物理10

密度与空心判断

一个铜球质量为178g，体积为30cm³。已知铜密度8.9g/cm³，判断铜球是否空心；若空心，空心体积是多少。

  1. 若铜球为实心，体积V实=m/ρ=178/8.9=20cm³。
  2. 实际体积为30cm³，大于同质量实心铜体积。
  3. 说明铜球是空心的。
  4. 空心体积V空=30-20=10cm³。

答案：铜球空心，空心体积10cm³。
