一次函数与方案选择

某研学活动有两种租车方案：甲方案固定费用300元，每人再收40元；乙方案无固定费用，每人收65元。设参加人数为x，费用分别为y甲、y乙。写出函数关系式，并判断人数在什么范围时选甲更省。

  1. 建立模型：y甲=300+40x，y乙=65x。
  2. 比较费用：令300+40x<65x，得300<25x，所以x>12。
  3. 因为x表示人数，必须取正整数，所以x≥13时甲更省。
  4. 当x=12时，甲费用780元，乙费用780元，两方案相同；x≤11时乙更省。

答案：y甲=300+40x，y乙=65x；13人及以上选甲更省。

圆切线与相似

如图，AB为圆O直径，点C在圆上，过C作圆的切线交BA延长线于P。证明：PC^2=PA·PB。

  1. 连接OC。因为PC是切线，所以OC⊥PC。
  2. 因为AB是直径，所以∠ACB=90°。结合P、A、B共线，可寻找△PCA与△PBC中的对应角。
  3. 由切线弦定理或角度转化可得∠PCA=∠PBC，又∠P公共，所以△PCA∽△PBC。
  4. 由相似得PC/PB=PA/PC，交叉相乘得PC^2=PA·PB。

答案：PC^2=PA·PB。

  3. 直线AB过A(3,6)和B(0,6)，两点纵坐标相同，斜率为0。
  4. 因此直线AB为水平直线，解析式y=6。

根的分布

关于x的方程 x^2-2(m+1)x+m^2-1=0 有两个正实根，求m的取值范围。

  1. 有两个实根，判别式Δ=[-2(m+1)]^2-4(m^2-1)=8m+8≥0，得m≥-1。
  2. 两根均为正，根和2(m+1)>0，得m>-1。
  3. 根积m^2-1>0，得m<-1或m>1。
  4. 综合m≥-1、m>-1、m<-1或m>1，最终m>1。

答案：m>1。

折叠最短路径

  4. 最短距离AD'=sqrt[(20)^2+6^2]=sqrt436=2sqrt109。

答案：PA+PD最小值为2sqrt109。

动点面积函数

直角三角形ABC中，∠C=90°，AC=8，BC=6。点P从C沿CA向A运动，CP=x，过P作PQ∥CB交AB于Q。求△APQ面积关于x的函数，并求最大值。

  1. 因为PQ∥CB，所以△APQ∽△ACB。
  2. AP=AC-CP=8-x，相似比AP/AC=(8-x)/8。
  3. △ACB面积为1/2×8×6=24，面积比等于相似比平方。
  4. S△APQ=24×[(8-x)/8]^2=3/8(8-x)^2。定义域0≤x≤8，此函数随x增大而减小，最大在x=0。

答案：S=3/8(8-x)^2，最大值24。

几何证明

平行四边形ABCD中，E、F分别为AB、CD中点。证明四边形AECF是平行四边形。

  1. 在平行四边形ABCD中，AB∥CD且AB=CD。
  2. E、F分别为AB、CD中点，所以AE=1/2AB，CF=1/2CD，因此AE=CF。
  3. 又因为AE在AB上，CF在CD上，而AB∥CD，所以AE∥CF。
  4. 一组对边平行且相等的四边形是平行四边形，所以AECF为平行四边形。

答案：四边形AECF是平行四边形。

交点与参数

直线 y=x+b 与抛物线 y=x^2-4x+3 有两个交点，且两交点横坐标之差为2，求b。

  1. 联立方程：x+b=x^2-4x+3，整理得x^2-5x+3-b=0。
  2. 设两交点横坐标为x1、x2，则x1+x2=5，x1x2=3-b。
  3. 已知|x1-x2|=2，所以(x1-x2)^2=4。
  4. 由(x1-x2)^2=(x1+x2)^2-4x1x2，得4=25-4x1x2，所以x1x2=21/4。
  5. 于是3-b=21/4，得b=-9/4。

答案：b=-9/4。
